# Documentación Web de Skell's GreenHouse

Esta carpeta contiene la documentación web estática del proyecto, accesible desde GitHub Pages o localmente abriendo `index.html`.

## Estructura
- `index.html`: Página principal con índice, instrucciones y detalles técnicos.
- `css/`: Estilos para la web.
- `js/`: Scripts para la web.

## Uso
Puedes navegar la documentación para conocer el hardware, software, instrucciones de uso y detalles de configuración del proyecto.

> Recuerda: para la pantalla TFT, reemplaza el archivo `User_Setup.h` de la librería TFT_eSPI por el incluido en `src/User_Setup.h`.
