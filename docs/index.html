Bienvenido a Skell's GreenHouse

Skell's GreenHouse es un invernadero inteligente y autónomo, desarrollado para monitorear y controlar variables ambientales usando ESP32, sensores y una pantalla TFT. El proyecto es open source y está documentado para facilitar su uso y adaptación.

Hardware

  • ESP32 Dev Board
  • Pantalla TFT ILI9341 (SPI)
  • Sensores: DHT22, DS18B20, sensor de humedad de tierra, sensor de nivel de agua
  • Luces UV (control PWM)
  • Bomba de agua
  • Placa y esquemas: Esquema KiCad

Software

  • Firmware principal: src/main/main.ino
  • Módulos: GUI, CONFIG, MONITOR
  • Servidor web integrado para monitoreo en tiempo real
  • Generación de interfaz gráfica en pantalla TFT
  • Documentación y código fuente en GitHub

Instrucciones de Uso

  1. Clona o descarga este repositorio.
  2. Abre el proyecto en Arduino IDE o PlatformIO.
  3. Configura tus credenciales WiFi en CONFIG_example.h y renómbralo a CONFIG.h.
  4. Reemplaza el archivo User_Setup.h en la carpeta de la librería TFT_eSPI por el incluido en src/User_Setup.h para asegurar la correcta configuración de la pantalla.
  5. Compila y sube el firmware al ESP32.
  6. Accede al servidor web desde la IP mostrada en la pantalla TFT.

Configuración de Pantalla TFT

El archivo User_Setup.h define los pines y parámetros para la pantalla ILI9341. Es imprescindible reemplazar el archivo original de la librería TFT_eSPI por el proporcionado aquí para el correcto funcionamiento del proyecto.

  • Pines SPI: MISO=19, MOSI=23, SCLK=18, CS=15, DC=2, RST=4, TOUCH_CS=21
  • Frecuencia SPI: 27 MHz
  • Driver: ILI9341
