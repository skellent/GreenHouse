Bienvenido a Skell's GreenHouse

Skell's GreenHouse es un proyecto dedicado a la creación de un sistema de invernadero inteligente. Nuestro objetivo es proporcionar una solución eficiente y sostenible para el cultivo de plantas, utilizando tecnología avanzada para monitorear y controlar las condiciones del invernadero.
